#### 使用步骤

1. 在 /temp 中下载符合平台的程序
2. 编辑配置文件 config.yaml 中的内容，修改要监听的理财产品
3. 手机上获取  PushDeer key，详见 [此处](https://www.pushdeer.com/)，IOS 系统使用相机扫描 **在线版AppCode** 即可。安卓需要下载 App。
4. 编辑配置文件 config.yaml 中的内容，修改 PushDeer key
